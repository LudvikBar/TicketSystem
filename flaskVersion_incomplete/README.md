TicketSystem



### **Start venv**
---

**For Command Prompt:**
```bash
.\venv\\Scripts\\activate.bat
```

**For Powershell:**
```bash
.\venv\Scripts\Activate.ps1
```

***In case you get permisson error***, give permission to the current user:
```bash
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
